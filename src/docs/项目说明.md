# 学生管理系统项目说明

本项目是一个基于 C++ 的学生信息管理系统，主要用于练习面向对象程序设计、STL 容器、文件读写和基础数据管理功能。

计划实现功能
1. 添加学生信息
2. 删除学生信息
3. 修改学生信息
4. 查询学生信息
5. 成绩排序
6. 平均分统计
7. 文件保存与读取
