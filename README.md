# student-management-system.cpp
A C++ student management system based on OOP and file persistence
# Student Management System C++

项目简介
这是一个基于 C++ 面向对象思想实现的学生成绩管理系统，主要用于练习类与对象、封装、STL 容器、文件读写和基础数据管理功能。

技术栈
C++,面向对象程序设计,STL vector,fstream 文件读写

核心功能
 添加学生信息, 删除学生信息,修改学生信息,查询学生信息,成绩录入与修改,按成绩排序,计算平均分,文件保存与读取
